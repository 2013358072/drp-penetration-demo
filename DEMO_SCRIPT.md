# DRP 穿透式监管演示 — 30 分钟讲解话术与时序

> 配合本仓库前端 Demo 使用：本地执行 `npm install` 与 `npm run dev` 后，按下列顺序全屏演示（建议 4K 大屏 + 笔记本双屏）。

## 0–3 分钟 · 开场冲击（全景驾驶舱）

1. 打开首页「全景监管驾驶舱」，停留 10 秒：强调 **82 家法人、5 大板块、1200+ 账户（滚动模拟）、在投项目、开放风险** 一屏统揽。
2. 指向 **风险热力图**：说明「板块 × 十大风险域」热度，政策要求的「四全穿透」先落在一屏态势上。
3. 点开顶部 **实时预警条**（自动轮播）：口述「不是事后报表，而是实时信号」。
4. 下拉到 **预置 10 类风险样本表**：点一行，现场跳转到对应穿透模块（投资/资金/采购等）。

**话术要点**：不讲技术架构，只讲「从看不见到看得见」。

## 3–7 分钟 · 投资穿透

1. 左侧菜单进入 **投资穿透**，说明五级横条：**集团 → 板块 → 子公司 → 项目 → 单据/工单**。
2. 在集团页点击某 **板块** 下钻 → 选 **子公司** → 选 **超预算/预警项目**（金额红色）。
3. 进入 **项目明细**：展示 **AntV G6 知识图谱**（投资链路 + 关联方），点击节点看高亮路径。
4. 点击 **进入单据 / 凭证层**：展示可研批复、完成投资月报、**核查工单 WO-2026-0037** 链接 → 可切到「核查工单」页闭环。

## 7–11 分钟 · 资金穿透

1. 进入 **资金穿透**：集团资金池饼图 + 动态折线（定时器模拟）。
2. 用 Drill 条进入 **板块 → 子公司 → 账户**；标红账户余额。
3. **流水/凭证层**：指出 **大额支付无合同引用** 行 → 点击「生成工单」跳转工单列表。

## 11–15 分钟 · 采购穿透

1. **采购穿透** 总览柱状图：AI 围标概率（×100）。
2. 五级下钻到 **标段 CG-2026-0501**：表格展示 **三家 IP/MAC 完全一致**。
3. 下方 **G6 关联图谱**：同 MAC、自然人参股关系。

## 15–18 分钟 · 合同穿透

1. **合同穿透** 下钻到高风险合同 **CT-2026-0312**。
2. **NLP 条款摘要**（无限责任 + 背靠背）+ **履约进度** + 合同关联 **G6 图谱**。

## 18–23 分钟 · 智能中台

1. **智能中台**：在问数框输入或点快捷 chip，如「有哪些大额异常支付？」。
2. 强调 **预置问答库 + 固定图表**，不依赖外网大模型。
3. 展示 **AI 预警推送列表** 与 **月度风险报告** 静态正文（可宣读要点）。

## 23–28 分钟 · 闭环

1. **风险预警中心**：按类型条形图 + 全表。
2. **核查工单**：状态色标（核查中/整改中/已销号）。
3. **整改销号**：点击 **推进到下一阶段**，演示流程灯。

## 28–30 分钟 · 价值升华

对比三句话收尾：**无 DRP 则穿透停留在抽样与报表；有 DRP 则全量监测、逐级穿透、工单闭环可证**。其余六个穿透模块为 **占位页**，可说明二期与四大模块同一交互范式。
