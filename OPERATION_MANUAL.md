# DRP 穿透式监管演示原型 · 操作手册

## 一、启动方式

```bash
cd /path/to/原型demo
npm run dev
# 浏览器打开 http://localhost:5173/
```

兼容：4K 大屏 + PC 浏览器，自动适配字号。

## 二、标准 30 分钟演示流程

### 0:00 ── 3:00 开场冲击：全景驾驶舱

| 操作 | 内容 | 话术要点 |
|------|------|----------|
| 打开首页 | 6 大 KPI 卡片（82 法人/1246 账户/42 项目…） | "这就是 DRP 全景驾驶舱，一屏看穿整个集团" |
| 页面自动弹预警弹窗 | 实时预警：工程建设板块围标串标 | "系统自动发现、主动推送，不用你想该查什么" |
| 向下滚动 | 板块 KPI 柱状图 + 风险热力图 | 移鼠标到热力图红色区域，突出风险集中度 |
| 点击风险样本表格任一行 | 自动跳转到对应的穿透模块 | "从驾驶舱到具体业务，一键穿透" |
| 看右侧预警轮播 | 饼图：37 条开放风险分布 | "重大风险 4 条、高风险 11 条，全在掌控中" |

### 3:00 ── 7:00 投资穿透

| 操作 | 内容 | 话术 |
|------|------|------|
| 左侧导航 → ① 投资穿透 | 集团投资总览 + 预算/实际对比柱状图 | "42 个在投项目，超预算自动标红" |
| 点击标红行（工程建设） | 下钻到板块 | "从集团总览下钻到业务板块" |
| 点击"海外工程总承包公司" | 下钻到子公司 | "锁定到具体子公司" |
| 点击"东南亚某国路桥 PPP" | 下钻到项目明细 | "项目级穿透：预算 12.4 亿，实际 14.7 亿" |
| 页面展示知识图谱 | 投资链路+关联方网络 | "知识图谱自动发现：当地分包与高管亲属企业同一控制" |
| 点击"进入单据/凭证层" | 可研批复 + 月度投资 + 核查工单 | "从宏观到微观，五级穿透到底" |
| 点击"打开工单列表" | 跳转核查工单页 | "发现问题直接生成工单，形成闭环" |

### 7:00 ── 11:00 资金穿透

| 操作 | 内容 | 话术 |
|------|------|------|
| 左侧导航 → ③ 资金穿透 | 资金全景池 KPI（可归集资金/大额笔数/拦截队列） | "资金全景池，每一分钱都在监控之下" |
| 注意 KPI 数字实时跳动 | 动态滚动效果 | "实时数据，不是静态报表" |
| 点击"下钻"到板块 | 各板块资金集中度 | 强调板块间的对比 |
| 点击"华东电力有限公司" | 账户明细 | "华东电力 2 个账户，A1002 已标红" |
| 点击标红账户"穿透流水" | 流水/凭证层 | "9200 万支付未关联合同编号，已自动触发拦截" |
| 点击"生成工单（演示）" | 跳转工单页 | "异常支付 → 拦截 → 核查 → 闭环" |

### 11:00 ── 15:00 采购穿透

| 操作 | 内容 | 话术 |
|------|------|------|
| 左侧导航 → ⑥ 采购穿透 | 采购招标全景，柱状图展示各标段围标概率 | "7 个在途标段实时监控" |
| 注意 CG-2026-0501 红色柱子 | AI 围标概率 0.94 | "这个标段，AI 判断围标概率 94%" |
| 点击板块 → 工程建设 → 海外工程总承包公司 | 按板块/公司过滤 | 快速演示下钻链路 |
| 点击 CG-2026-0501 | 投标文件比对 | "三家投标单位：IP 相同、MAC 相同" |
| 指出表格关键字段 | IP: 103.21.88.12 完全一致 / MAC: A4:83:E7:11:22:33 | "同一台电脑投了三份标书" |
| 报价列 | 2980 / 3010 / 3040 | "标准阶梯分布，围标经典模式" |
| 知识图谱 | 三家 + 自然人王某的股权关联 | "三家都与同一个自然人有关联，围标概率 0.94" |

### 15:00 ── 18:00 合同穿透

| 操作 | 内容 | 话术 |
|------|------|------|
| 左侧导航 → ⑤ 合同穿透 | 合同总览图表 | "NLP 自动解析合同条款" |
| 点击下钻到"城市基建投资公司" | 合同列表 | 点 CT-2026-0312 |

### 18:00 ── 23:00 智能体赋能（重点演示）

| 操作 | 内容 | 话术 |
|------|------|------|
| 左侧导航 → 智能中台 | 5 个 Tab 页 | "AI 智能体群，从被动工具到主动监管助手" |

#### Tab 1: 智能问数（核心亮点）
| 点击"新能源板块本季度投资回报率是多少？" | AI 流式思考→流式回答→展示图表 | "注意看——AI 不是直接给答案，而是展示思考过程" |
| 思考过程逐行出现 | 意图解析→调用数据中台→数据返回→生成图表 | "每一个步骤都透明可追溯，不是黑盒" |
| 回答字符逐个输出 | 打字机效果 | "流式输出，就像真人在跟你对话" |
| 再问第二个问题 | 围标串标相关 | "再展示一次——从问题到答案只需几秒" |

#### Tab 2: 主动预警
| 切换到"主动预警" Tab | 3 条 AI 主动推送 | "数找人——不用你想该查什么，系统主动告诉你" |
| 点击第一条的"查看关联图谱" | 自动跳转到关联分析 Tab | 展示跨 Tab 联动 |
| 点击"发起核查工单" | 跳转工单页 | 预警→处置一条龙 |

#### Tab 3: 关联分析
| 输入"某设备贸易商行" | 知识图谱展开过程（流式思考） | "输入名字，秒级呈现全部关联关系" |
| 看图谱中红色路径 | 张某 → 控股 → 某设备贸易商行 → 华东电力 | "前采购副经理控股供应商，利益输送链一目了然" |

#### Tab 4: 报告生成
| 点击"生成本月风险分析报告" | 流式思考→报告逐段输出→图表 | "AI 自动汇总、自动分析、自动生成结构化报告" |

#### Tab 5: 决策建议
| 点击"分析近6个月数据并给出建议" | 流式分析→输出 2 条等级调整建议 | "AI 不是只告诉你数据，而是直接给你决策建议" |
| 强调海外工程公司 | 橙色→红色，14 次预警 | "AI 建议主动调升风险等级，并给出处置动作" |

### 23:00 ── 28:00 闭环处置全流程

## 五、关键演示技巧

1. **AI 流式效果**：提前打开智能中台页，第一次问答会有思考过程+打字输出，最震撼
2. **热力图互动**：驾驶舱热力图 hover 效果很好，演示时多用
3. **知识图谱拖拽**：G6 图可以拖拽、缩放，看起来很有科技感
4. **整改进度条**：整改销号页的"推进"按钮是交互亮点
5. **对比页面收尾**：最后一看对比表，决策者印象最深
